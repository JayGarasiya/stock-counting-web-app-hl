# Stock Counting Web App

A powerful Stock Counting and Inventory Management Web Application built using nopCommerce and ASP.NET Core.

---

## Features

### Inventory Management
- Stock Tracking
- Product Quantity Management
- Inventory Updates
- Warehouse Stock Monitoring

### Product Management
- Add Products
- Edit Products
- Delete Products
- Product Categorization

### Admin Dashboard
- Inventory Overview
- Product Statistics
- Order Monitoring
- Reports & Analytics

### User Management
- Admin Authentication
- Role Management
- Customer Management

### Order Management
- Order Tracking
- Order Status Updates
- Invoice Management

---

## Technologies Used

### Backend
- ASP.NET Core
- nopCommerce
- Entity Framework Core
- SQL Server

### Frontend
- Razor Views
- Bootstrap
- jQuery
- HTML5
- CSS3
